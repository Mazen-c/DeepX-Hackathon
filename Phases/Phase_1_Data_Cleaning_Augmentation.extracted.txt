TASK DELEGATION
Person A · Cleaning pipeline
pip install pyarabic pandas — both are tiny, fast installs.
Normalize Alef variants: re.sub(r'[أإآ]', 'ا', text)
Normalize Taa Marbouta: re.sub(r'ة', 'ه', text)
Strip Tashkeel: araby.strip_tashkeel(text)
Strip Tatweel: araby.strip_tatweel(text)
Reduce repeat chars: re.sub(r'([\u0600-\u06FF])\1{2,}', r'\1', text)
Emoji-to-Arabic dict: hardcode top 25 emoji (no library needed).
Output: train_clean.csv, val_clean.csv, test_clean.csv.
Person B · Augmentation pipeline
Count class distribution: train_clean.groupby(['aspect','sentiment']).size()
Identify all (aspect, sentiment) pairs where count < 100.
Use Groq free API (llama-3-70b-8192) to generate 30 synthetic examples per deficient pair.
Batch requests: 5 examples per call to stay under token limits.
Back-translation via Helsinki-NLP/opus-mt-ar-en (48MB) + opus-mt-en-ar (48MB) — download these two only.
Output: train_augmented.csv with class distribution printed to console.

GROQ AUGMENTATION — RATE LIMIT STRATEGY
Groq free tier limits: 30 requests/min, 14,400 requests/day, 6,000 tokens/request. Budget your calls before writing a single line.

import time, os
from groq import Groq
client = Groq(api_key=os.environ['GROQ_API_KEY'])

def gen_batch(aspect, sentiment, n=5):
    prompt = f'''Generate {n} short Arabic restaurant reviews where
    the ONLY aspect mentioned is {aspect} with {sentiment} sentiment.
    Return ONLY a JSON array of strings. No preamble.'''
    resp = client.chat.completions.create(
        model='llama-3-70b-8192',
        messages=[{'role':'user','content':prompt}],
        temperature=0.9, max_tokens=400
    )
    time.sleep(2.1)   # Stay under 30 req/min hard cap
